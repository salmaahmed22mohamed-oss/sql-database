#  Car Rental System

A relational database system designed to manage car rental operations efficiently. Built using Microsoft SQL Server with a focus on proper schema design, normalization, and SQL query optimization.

---

##  Overview

The Car Rental System is a database project that handles the core operations of a car rental business, including managing car inventory, customer records, rental transactions, and payment tracking.

---

## Features

-  Car inventory management (available, rented, maintenance)
-  Customer registration and profile management
-  Rental booking and return tracking
-  Payment records and billing
-  Availability checking for specific date ranges

---

##  Database Schema

| Table | Description |

| `Cars` | Stores car details (model, brand, year, status) |
| `Customers` | Stores customer information and contact details |
| `Rentals` | Tracks rental transactions (dates, car, customer) |
| `Payments` | Records payment details for each rental |

---

##  Technologies Used

- Microsoft SQL Server
- SQL (DDL, DML, Joins, Stored Procedures)
